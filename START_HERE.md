## 💡 How It Works (Super Simple)

**Every Sunday:**
- Everyone sets preferences (cuisine, price, duration)

**Monday 9 AM:**
- App checks calendars, picks central location, finds 2 restaurants

**Monday/Tuesday:**
- Everyone votes on their favorite

**After all votes:**
- App picks winner, shows on Dashboard

**Lunch day:**
- Show up and enjoy! 🍽️

## ❓ Common Questions

**Q: Can I test it first before deploying?**
Yes! Follow [GETTING_STARTED.md](GETTING_STARTED.md) to run locally.

**Q: How much does it cost?**
$0/month for 5-9 users. Railway + Vercel free tiers are plenty.

**Q: Do I need to keep my computer on?**
No! Once deployed to Railway/Vercel, it runs in the cloud 24/7.

**Q: Can friends access it from anywhere?**
Yes! They just need the URL and internet connection.

**Q: What if I change the code later?**
Just `git push` and it auto-deploys. No manual steps.

**Q: How do I update if it breaks?**
Railway and Vercel have logs. Check those first.
