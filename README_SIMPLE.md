# Lunch Picker - Simple Guide

## What You Have

A complete web app for coordinating weekly lunches with 5-9 friends in SF.

## What It Does

1. Everyone sets food preferences each week (30 seconds)
2. Monday morning: App automatically generates 2 restaurant suggestions
3. Everyone votes (10 seconds)
4. Winner is automatically selected
5. Show up for lunch! 🍽️

## How Your Friends Access It

Since you don't share WiFi, you need to deploy online.

### ⚡ Easiest Way: Railway + Vercel (10 min, $0/month)

**Read:** [QUICK_DEPLOY.md](QUICK_DEPLOY.md)

**Steps:**
1. Push code to GitHub
2. Deploy backend to Railway.app (backend + database)
3. Deploy frontend to Vercel.com (frontend)
4. Share the Vercel URL with friends

**Result:** 
- You get a URL like `https://lunch-picker.vercel.app`
- Friends access it from anywhere
- Works 24/7 in the cloud
